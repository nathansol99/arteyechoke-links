# Art Eye Choke - Link Page

Link-in-bio page for Art Eye Choke. Hosted on Cloudflare Pages.

## Live Site

https://links.arteyechoke.com

## How to Edit Links

1. Click on `index.html` in this repository
2. Click the pencil icon (Edit this file)
3. Find the link you want to change
4. Edit the URL or text
5. Click "Commit changes"
6. Site updates automatically in about 30 seconds

## Adding a New Link

Copy this template and paste it where you want the new link to appear:
```html
<a href="YOUR_URL_HERE" class="link-button" target="_blank" rel="noopener">
    <span class="icon">ðŸ”—</span>
    Your Link Text
</a>
```

Replace `YOUR_URL_HERE` with the actual URL and update the text/emoji.

## Removing a Link

Delete the entire `<a>...</a>` block for the link you want to remove.
